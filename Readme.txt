How to Run

first code the root file in cmd
 
then run the below command

  cd server
  composer install
  php artisan optimize
  php artisan serve

the create new terminal tab run the below commands

    cd client
    npm install
    npm run dev



key features

- add new employee 
- notify the user with when there is  payment
- fully customable allowance system
- excel export
- role based access between finance officer and head finance officer
- adding of allowance ( the rule that declares until what value it tax free) and allowance value ( how much the employee paid)
- payroll history and filter by month
 - dashboard to visualize current month payroll summery
 - a fully working authentication with route guard  and more...
